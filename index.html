CAPABILITIES

Built for Real-World Fraud Detection

Every component of AEGIS-X is engineered for production-grade UPI security, combining classical machine learning with real-time network telemetry.

Three-Model Ensemble

Combines Random Forest, Support Vector Machine, and Logistic Regression in a voting ensemble for robust, low-false-positive classification.

Real-Time Inference

Flask-powered ML server delivers fraud predictions in under 14 milliseconds — fast enough to block fraudulent transactions before they complete.

Geo-Anomaly Detection

Automatically flags transactions originating from high-risk international IP ranges (RU, CN, NG, PH, RO, UA) with geo-velocity analysis.

UPI Registry Verification

Cross-references each VPA against a curated registry of 200+ verified UPI identities. Unregistered IDs are instantly flagged for review.

PCA Feature Engineering

28 principal components (V1–V28) extracted via PCA from raw transaction data, preserving variance while protecting user privacy.

Live SOC Console

A real-time Security Operations Console streams live transaction decisions, inspector view, ML pipeline visualisation and system metrics.

SYSTEM DESIGN

End-to-End Architecture

A full-stack fraud detection pipeline — from raw UPI transaction to a real-time blocked/verified decision.

01
UPI Transaction
A UPI payment is initiated. The VPA, amount, location, IP, and timestamp are captured by the pipeline.
02
Node.js Backend
Runs on port 3001. Validates the VPA against the UPI registry, applies geo-anomaly rules, then forwards to ML service.
03
Flask ML Server
Runs on port 5000. Runs the three-model ensemble on PCA features. Returns isFraud + riskScore in ~14ms.
04
Verdict
Transaction is either VERIFIED (safe) or BLOCKED (fraud). The Vite frontend streams results in real-time to the SOC console.
